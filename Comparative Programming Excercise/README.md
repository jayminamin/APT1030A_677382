# Student Registration Checker

## Purpose
This tool automates student schedule checks. You enter a student name and total registered units. The system compares the unit count against standard academic limits. Exceeding the limit requires administrative overload approval to proceed.

## Running the Code

### Python
You need Python 3 installed. Type python script.py or python3 script.py in your terminal to start.

### Java
Grab the Java Development Kit first. Type javac RegistrationChecker.java to compile the code then type java RegistrationChecker to run it.

### JavaScript
Node and the readline sync module are required here. Type npm install readline sync then type node script.js to start.

### C
Set up a C compiler like gcc. Type gcc script.c to build the file. Run ./a.out on Linux or Mac. Run a.exe on Windows.

## What does the project demonstrate?
The project highlights fundamental differences between distinct programming paradigms.
The code contrasts the minimalist approach of Python with the boilerplate demands of Java. Java requires you to write classes and methods and imports.
C requires you to manually define the array size. High level languages like Python and JavaScript manage string memory automatically.
You see various ways to capture user data. Python uses a simple input command. Java requires you to build Scanner objects. C uses the fgets function.
The if and else logic remains identical across all languages. The project shows how Java and JavaScript feature ternary operators to replace standard blocks.
