# Level 7 - Capstone Implementation

## Context: Design a Mini DSL for Mobile Money Rules

Students must:
1.	Define a simple rule syntax:
    Example:
    TRANSFER 5000 FROM A TO B IF BALANCE > 1000

2. Write:
    * Parser in Python
    * Interpreter logic

## Implementation Requirements
1.	Parse command string.
2.	Extract:
    * Amount
    * Sender
    * Receiver
3.	Validate balance.
4.	Print transaction result.

## Reflection Component
Students must justify:
* Why Python is suitable for DSL prototyping.
* Why C++ would be harder for rapid rule iteration.
* How AI code assistants could accelerate DSL development in Kenya.
