# Level 3 — Control Flow and Decision Systems

## Context: Drought Advisory Logic (Machakos County)

Concepts
* if-else
* nested conditions
* loops

## Implementation Task

Write a rainfall advisory program:

Input:
* Rainfall amount (mm)
* Temperature

Rules:
* If rainfall < 200mm → "Irrigation Required"
* If rainfall ≥ 200mm and temperature > 30°C → "Monitor Soil"
* Else → "Normal Conditions"

Implement in:
* Python
* C++ or Java

## Extension Task

Rewrite logic using:
   * Nested if statements
   * Switch (Java/C++ where appropriate)

Compare readability.
